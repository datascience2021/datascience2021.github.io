Installing and testing a keylogger written in C#.


Step 1:    Find out the version of the .NET Framework installed on your computer by typing:     dir /a:d c:\Windows\Microsoft.NET\Framework64    on the command prompt.

Step 2:     If you already have the .NET Framework version 4.0.30319 you can go to step 4. FYI, you could have determined the .NET Framework version by doing what is shown on this article.

Step 3:     Download the .NET Framework from here and install it. Verify your installation by doing step 1.

Step 4:     Create a folder named C:\Keylogger and a file named svchost.txt within that folder. Go to this link and copy the text shown to the svchost.txt file and save it.

Step 5:     Go to the command prompt and type:

cd c:\Windows\Microsoft.NET\Framework64\v4.0.30319


csc /out:C:\Keylogger\svchost.exe C:\Keylogger\svchost.txt


cd C:\Keylogger


dir --- to verify the creation of the executable, but do not run it yet.


tasklist | find /i "svchost" >> mylist.txt --- to find out all running svchost processes so far.


svchost.exe --- to run the keylogger.


Step 6:     Now we need to type something to show how the keylogger is logging our keystrokes. Go to http://www.allitebooks.com and search for a book about memory forensics. Remember the idea is to type whatever.

Step 7:     Go back to the command prompt (the command line interface or CLI) and type:

tasklist | find /i "svchost"


Find the out the Process ID of the running keylogger. You could compare the output with the previous list you saved before (mylist.txt).

Step 8:     Download the Process Explorer (by Mark Russinovich) from right here and run it. Find the svchost running the keylogger, right click it and go to the properties (notice the path, the started). Kill the process. Verify that it's not running by typing again tasklist

Step 9:     Go to Virus Total and submit the svchost.exe to see what can their AV Engines find out about the keylogger.
