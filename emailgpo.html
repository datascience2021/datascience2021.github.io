Creating an email application with C# and using it via a local policy.



    Step 1.  

      Go to http://www.throwawaymail.com/ and keep the window open. You'll soon get emails on that address. If you want you could use https://www.guerrillamail.com/ instead of http://www.throwawaymail.com/.


    Step 2.  

      Create a folder named C:\email and a file named email.txt within that folder.

    Step 3.  

      Go to this link, copy the text and save it to the email.txt file.


    Step 4.  

      Open the email.txt file with Notepad and replace slalenisli@throwam.com with the temporary email address you got.


    Step 5.  

      On the command prompt, type the following commands to compile the C# code into a .EXE file.

cd c:\Windows\Microsoft.NET\Framework64\v4.0.30319


csc /out:C:\email\email.exe C:\email\email.txt


C:\email\email.exe



    Step 6.  

       Verify that the email was received.



    Step 7.  

      Go to https://www.virustotal.com/ and analyze the email.exe file. What info was found out about the file?


    Step 8.  

      Now we'll create a group policy using gpedit.msc. The GPO will run a logon script that sends an email every time a user logs in. To do that, first copy the c:\email\email.exe file to C:\Windows\System32\GroupPolicy\User\Scripts\Logon . The picture below shows how to create the local policy. And don't forget to update the policy by typing gpupdate /force and to verify its existence by typing: gpresult /R




    Step 9.  

      Log off and then log back in. Check if your temporary email account got a message.
