Scheduling tasks on Windows.


    Step 1.  

      Go to the command prompt and type: schtasks.exe. To keep a record of the actual tasks type this: schtasks > tasks.txt. Open the tasks.txt with Notepad, read the tasks and close it. Did you realize how many tasks have been scheduled?



    Step 2.  

      Go to the command prompt and write the code shown below.


mkdir C:\Reports

for /L %i in (1,1,100) do echo "Where's the money?" > C:\Reports\file%i.txt

dir C:\Reports

cd %homepath%

echo del /s /q C:\Reports\*.* > nomorereports.bat

taskschd


    Step 3.  

      Create a basic task (that runs the nomoreports.bat batch file) to delete all the files in the C:\Reports folder five minutes from your computer's actual time. The pictures below show how to do it on Windows 10, but it's quite the same on Windows 7. Type taskschd on the command line to get started.




    Step 4.  

      Repeat Step 1. Is the new task listed?


    Step 5.  

      Wait until the task is run and verify if the C:\Reports files are gone. Did it work?
